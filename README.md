# File Handling using Object-Oriented Programming (OOP)

A Python-based project demonstrating Core Object-Oriented Programming (OOP) concepts integrated with robust file handling mechanisms.

## ey Features
* **Encapsulation & Inheritance:** Structured data models using classes for clean data management.
* **File Operations:** Implementation of structured data reading, writing, and updating using Python built-in modules.
* **Modular Design:** Divided into execution scripts (`driver_main_oops_python.py`) and module configurations (`org_emp_oops_python.py`).

## Repository Structure
* `driver_main_oops_python.py` - Main execution driver script.
* `org_emp_oops_python.py` - OOP class structures (Organization/Employee logic).
* `filehandling_without_class_python.py` - Baseline functional file handling script.

## 🛠️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project directory and execute:
   ```bash
   python driver_main_oops_python.py
   ```
